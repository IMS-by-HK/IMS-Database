# IMS-Database
Mongo Database for Inventory Management System



---
Initalising/dependencies installed:
- npm init -y
- npm install mongoose express
- npm install --save-dev nodemon
- npm install dotenv

----
Features:
- Localized product descriptions
    - set to english
- Product model, schema & controller
- UserModel & Schema

---
Techstack:
Express
Mongoose
CloudAtlasDB
